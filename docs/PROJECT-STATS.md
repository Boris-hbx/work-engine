## Features & Modules

### Core Features

1. **Todo Management** - Four-quadrant task management system
2. **Prompt Log** - AI prompt logging and management
3. **Prompt Tasks** - Execute and track prompt-based tasks
4. **Bubble Charts** - Interactive bubble chart visualization
5. **Motivation** - Daily motivational quotes and content
6. **Games** - Recreational game collection

### Games Available

1. **Spider/You Come Here (你过来啊)** - Territory capture game
2. **Breakout (各个击破)** - Classic brick breaker
3. **Zen (发呆时光)** - Relaxation animations
4. **Tower Defense (TRC攻防大师)** - Strategic tower defense

### Tower Defense Features

- 7 tower types with unique abilities
- 7 city maps (Canadian cities theme)
- Skill system (Meteor, Freeze, Slow Time)
- Achievement system
- Talent tree system
- Daily challenges
- Endless mode
- Leaderboard
